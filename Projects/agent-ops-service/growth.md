# agent-ops-service — 그로스 (growth)

`growth-marketer`가 채널 실험·카피·결과를 누적. 무료 채널만.

## 채널 (개발자 대상, 무료)
- **1순위 — 디스콰이엇 / GeekNews**: 메이커·개발자 커뮤니티. build-in-public 문화 → "AI 직원으로 1인 회사 굴리기" 일지가 자연스럽게 읽힘.
- **2순위 — 요즘IT / OKKY**: 한국 개발자 기고·커뮤니티. 안전·자동화 노하우 글로 인바운드.
- **3순위 — 링크드인**: 1인 SaaS 운영자·AI 에이전시 ICP 콜드 DM(진단 결과 선제공 시).

## 콘텐츠 → 서비스 깔때기
콘텐츠가 청중·신뢰 → 그중 "나도 무인 운영하는데 안전 걱정"인 사람이 무료 진단 신청 → 유료 정비/리테이너. 콘텐츠 끝 CTA는 항상 무료 진단 오퍼.

## 카피 초안 (1호 글)
- 제목(안): "AI 직원 7명에게 내 창업 아이디어를 맡겼더니, 2주 검증을 만들기도 전에 다 죽였다"
- 훅: 직원 대신 에이전트로 1인 회사를 굴리는 실험 — 비판가 에이전트가 6개 아이디어를 전부 깬 과정과, 그게 왜 돈을 아낀 건지.
- CTA: "Claude Code를 무인 cron으로 돌리는데 범위 밖 커밋·시크릿 유출이 걱정되면 무료 30분 안전 진단 — 댓글/DM."
