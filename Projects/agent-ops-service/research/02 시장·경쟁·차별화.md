> 2026-06-08 실행 회의(agent-ops-execution-council) 시장 리서처 산출. 출처는 말미.

## 유효 모수 (가장 약한 고리)
- 명목상 수만(국내 순수 개발자 ~17.3만·SW전문인력 ~35.6만, SPRi). 1인 창업 부근으로 좁히면 더 작다.
- Claude Code 한국 사용자 **다수가 비개발자** — 패스트캠퍼스/AI싱크클럽 설문 300명 중 개발자 20%뿐.
- "에이전트를 코드로 무인 운영"하는 사람은 그 안의 소수. 국내 실전 사례 블로그는 손에 꼽음.
- **추정: 유효 초기 유료 모수 = cron/hook 무인 운영 개발자(수천 추정) × 안전에 돈 낼 전환율(한 자릿수 %) → 연 수십~수백 건 유료 진단이 현실적 천장.** 부트스트랩 1인 매출엔 가능하나 모수가 본 트랙의 가장 약한 고리.

## 인접 경쟁
- **패스트캠퍼스 '하네스 엔지니어링' 강의**(₩27만, 정가 ₩45만) — 가장 위협적. permission·sandbox·권한관리·pre-commit hook·멀티에이전트 하네스를 이미 다룸. **약점**: 셀프 학습 일반론, "내 환경 1:1 진단·롤백"은 안 해줌, 일회성.
- **claudecode.to** — 워크숍+코칭+AX 컨설팅, 무료 60분→유료(동일 깔때기). **약점**: 엔터프라이즈/부서 대상(1인·소팀과 충돌 안 함, 상보적), 안전은 한 줄 언급뿐.
- **AI 자동화 컨설팅 일반**(하마다랩스·SK AX 등) — 비즈니스 워크플로 자동화지 코딩 에이전트 무인 안전이 아님.
- **비개발자용 강의** — 청중이 기획·마케터라 깊은 기술 니즈와 어긋남.
- **공통 약점 = 우리가 칠 공백**: 강의는 일반론·셀프, 컨설팅은 엔터프라이즈·고가. **"1인/소팀 개발자에게 자기 레포를 보고 안전 가드를 직접 깔아주는 한국어 1:1 정비"는 비어 있다.**

## 차별화 (안전 지식이 아니라 결합)
1. **복제 불가한 증거**: claude-radar·automation-safety·계약 테스트를 실제 6개월 운영. 강의는 "이렇게 하세요", 우리는 "내가 깨먹고 막은 1차 사례"(build-in-public이 증거).
2. **1:1 환경 맞춤**: 고객 실제 레포·settings.json·cron을 보고 권한 와일드카드·시크릿·롤백 부재를 그 환경에서 고침(강의는 진단 못 함).
3. **검증 가능 산출물**: before/after — 범위밖 변경 자동 롤백 동작 + 계약 테스트 통과 + 시크릿 0건. 1인 신뢰 문제 우회.
4. **좌표 공백 + 한국어 비대칭**(OKKY·GeekNews 신뢰 회로).

## 인접 확장 기회
진단 체크리스트 부트킷화 · 무인 운영 모니터링 SaaS-lite(설정 정기 스캔) · build-in-public 콘텐츠 자산화(뉴스레터/강의) · 소형 에이전시 화이트라벨 · (피벗) Routines 시대의 권한·승인 정책 설계 컨설팅.

## 가장 강한 리스크 (트랙을 죽일 수 있는 두 사실)
1. **Anthropic이 무인 운영+가드를 native 흡수 중** — `/loop`(내장 유지보수 안전 프롬프트)·`CronCreate/List/Delete`·`CLAUDE_CODE_DISABLE_CRON` 킬스위치·7일 자동만료·**Routines**(관리 인프라 무인 실행)·**Channels**(CI 실패 push)·GitHub Actions. 우리가 직접 짠 cron 래퍼·fetch-guarded push 동기를 공식이 메우는 중. (단 Routines는 "Permission prompts: No"라 오히려 가드 제거 → 안전 우려는 정당화하나 "공식이 그렇게 굴리는데 왜 돈 내고 가드?" 반론도 키움.)
2. **에이전트 신뢰성의 구조적 한계** — 8단계×85%≈27% 성공의 cascading failure. 가드 12개를 깔아도 근본 실패율은 못 낮춤 → "가드 깔았는데 또 터졌다"는 평판 리스크.
- **결론**: "안전하게 무인 운영" 명제가 (a)공식 기능 흡수 (b)신뢰성 구조적 한계로 약속을 다 못 지킴 — 이 둘이 모수 얇음보다 트랙을 더 빨리 죽일 수 있다. → 30일 안에 **₩15만 1건 결제**라는 가장 싼 수요신호부터 확인할 것.
